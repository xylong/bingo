基于go语言gin框架的web项目骨架，专注于前后端分离业务场景

### 安装
```
go get -u github.com/xylong/bingo
```

### 功能
- [x] 请求参数
    - [x] 自动映射参数
    - [x] 自动校验
- [x] 路由
    - [x] 分组
    - [ ] 限流
- [x] 中间件
    - [ ] 全局中间件
    - [x] 分组中间件
    - [ ] 路由中间件
- [ ] 日志
    - [ ] 日志分割

### 相关库
- [gin](https://github.com/gin-gonic/gin) http框架
- [logrus](https://github.com/sirupsen/logrus) 日志
